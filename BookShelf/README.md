# BookShelf - Inventory Management System
Simple C# web application tracks the the Inventory of books published.

## Mode Attributes of a Published Book
1. BookISBN
2. Title
3. Author
4. Published Date
5. Quantity

## Project Components
1. Utilized Bootstrap
2. MVC Model
3. Validation
4. CRUD Operation
5. CRUD Users
6. Assign Roles to Users
